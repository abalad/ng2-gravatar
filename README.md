# ng2-gravatar-directive

Angular2 gravatar directive

## Install

    npm i --save ng2-gravatar-directive

## Import

    import {Gravatar} from 'ng2-gravatar-directive';

## Usage

    @NgModule({
        declarations: [ Gravatar ]
    })
    
    <img gravatar [email]="user.email" [size]="16">
